Responsive Portfolio Website

Overview

A responsive personal portfolio website built using HTML, CSS, and JavaScript. The project showcases skills, projects, and contact information while following accessibility and responsive design best practices.

Features

- Responsive Design
- Dark Mode Toggle
- Accessibility-Friendly Contact Form
- Skills Section
- Case Study Projects
- Resume Download Button
- Mobile-Friendly Layout

Technologies Used

- HTML5
- CSS3
- JavaScript

Projects

1. Event Registration Portal
2. AI Travel Planner
3. Travel Management System

Setup

1. Clone the repository
2. Open the project folder
3. Open index.html in a browser

Reflection

This project helped me improve my understanding of responsive web design, accessibility, semantic HTML, and front-end development best practices.
